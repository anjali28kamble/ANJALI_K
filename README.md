# Anjali Kamble Portfolio

A personal portfolio website built using React, Vite, Tailwind CSS, Node.js, Express, and MongoDB.

## Features

* Responsive Design
* Modern UI/UX
* Projects Showcase
* Skills Section
* Contact Form
* Resume Download

## Tech Stack

* React
* Vite
* Tailwind CSS
* Node.js
* Express.js
* MongoDB

## Installation

```bash
git clone https://github.com/anjali28kamble/ANJALI_K.git
cd ANJALI_K
npm install
npm run dev
```
